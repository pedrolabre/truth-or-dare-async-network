# 🎨 Apresentação dos Protótipos de Interface

Esta pasta contém os protótipos de tela estáticos em HTML/CSS desenvolvidos para guiar o desenvolvimento e definir a especificação visual inicial da plataforma **Truth or Dare Async Network**.

Os protótipos cobrem os fluxos principais do aplicativo e estão divididos em duas versões de design:
- **Modo Escuro (Dark Mode)** localizado no diretório [dark-mode/](./dark-mode/)
- **Modo Claro (Light Mode)** localizado no diretório [light-mode/](./light-mode/)

---

## 🎨 Especificações Técnicas de Estilo

Para mais detalhes sobre as paletas de cores e tokens do sistema de design Tailwind, consulte o código das telas na pasta [dark-mode/](./dark-mode/) e [light-mode/](./light-mode/).
